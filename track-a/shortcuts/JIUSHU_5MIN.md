1. 打开 **快捷指令** → 右上角 **+** → 名称 **`主见`**
2. 按 [`SETUP.md`](./SETUP.md) 添加 8 个动作（或 Moses 已做好的 `.shortcut` 文件 **分享 → 九叔 iPhone**）
3. 两处必改：
   - POST 的 **URL** → `https://keel-production-be1c.up.railway.app/v1/entry`
   - Header **`X-API-Key`** → Moses 给的 key（与 GitHub/Railway 一致）

**主屏**：快捷指令详情 → **⋯** → **添加到主屏幕** → 图标名 **`主见`**。
